Airfoil Definition

The purpose of the Airfoil Definition dialog box is to study and modify a set of Airfoils, which are used in the Blade Definition dialog box to define the blades of wind turbine model. An Airfoil is basically a cross section of the blade where the different aerodynamic properties are specified.

FEDEM WindPower uses the same airfoil file format as AeroDyn. The Airfoil dialog box is thus simply an editor for such files. See the AeroDyn documentation (nrel.gov) for information on this file format.

A set of sample Airfoils (provided by NREL) is included with the FEDEM Windpower package (see the Properties\AeroData sub-folder of the installation). In some projects, the user may be less concerned with the blade designs, and most of these Airfoils will be sufficient. But, in other projects, the user may wish to provide their own Airfoils.

On the left side of the Airfoil definition dialog box, some global definitions are given:

  • Name - Provides the name of the Airfoil file.
  • Number of airfoil tables in this file - Airfoil files are designed so that they can contain many airfoils. The Airfoil file in the figure above contains a single Airfoil table.
  • Table ID parameter - Provides an identifier for each Airfoil table.
  • Stall angle [deg] - Provides the stall angle (in degrees).
  • Zero Cn angle of attack [deg] - Provides the zero-lift angle-of-attack of the AirFoil. Only used for Beddoes dynamic stall model.
  • Cn slope for zero lift (dimensionless) - Provides the static Cn curve slope near zero lift.
  • Cn extrapolated to value at positive stall angle of attack - Provides the Cn value at positive static stall. This is the nominal stall for positive and increasing angles-of-attack.
  • Cn at stall value for negative angle of attack - Provides the Cn value at the negative static stall angle of attack.
  • Angle of attack for minimum CD[deg] - Provides the angle of attack for the minimum drag coefficient.
  • Minimum CD value - Provides the minimum drag coefficient of the AirFoil.
On the right side of the Airfoil definition dialog box, The Airfoil table data (lift and drag coefficients) is given:

  • alpha - This is the angle of attack. The table must be written in order of increasing angle of attack. It is preferable to use a table for angles in the range -180 to +180 degrees. The coefficient values should be the same at -180 and +180 degrees to avoid discontinuity.
  • CL - Static lift coefficient.
  • CD - Drag coefficient.
  • CM - Pitching moment coefficient.

The Add Row and Delete Row buttons can be used to add and remove rows in the table.
